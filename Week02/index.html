Here is a form containing basic input types. For more info see W3 schools tutorials . Whenever you need to make an anchor tag, but don't have the final url, use #!

Simple Text Input

The box and title seen are created by fieldset and legend tags. Think modern day form groups/input groups.


TextArea Tag

Textarea tags is great for longer text answers, such as sentences and paragraphs


Checkboxes

Checkboxes allow the user to select more than one value (checkbox) at once. Checkboxes in the same group have the same name but different pre-defined values.






Radio Buttons

Radio buttons allow the user to only select one value at once. Radio buttons in the same group have the same name but different pre-defined values.





Select Options (Dropdown Menu)

Using the select and option tags, you can add items to a dropdown selection

Finally, we use a button to submit our form!
